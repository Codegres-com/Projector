# Quotation Generation

The Quotation Generation module focuses on transforming project estimates into professional, client-facing documents. It handles the formatting, branding, delivery, and formalization of the agreement, ensuring that the agency presents a polished image to the client.

## Feature List

| Feature Name | Description | Priority | User Benefit |
| :--- | :--- | :--- | :--- |
| **PDF Export** | Generate high-quality PDF documents of the quotation. | **High** | Essential for sharing uneditable, professional documents with clients. |
| **Document Templates** | Select from various visual layouts and branding styles (e.g., Modern, Corporate, Minimal). Allows customization of colors, fonts, and logos. | **High** | Ensures brand consistency and allows tailoring the look and feel to the specific client or project type. |
| **Iron Triangle Disclosure** | Clearly displays the selected constraint configuration (e.g., "Fixed Price & Scope; Flexible Timeline") on the generated PDF/Proposal. | **High** | Manages client expectations and protects the agency by transparently communicating project risks and flexibility. |
| **Digital Signatures** | Integration with e-signature providers or a built-in signing mechanism to allow clients to legally sign and accept the quotation online. | **Medium** | Speeds up the approval process and reduces paperwork. |
| **Email Integration** | Send the quotation directly from the platform to the client, with customizable email templates and tracking (opened/viewed). | **Medium** | Streamlines the delivery process and keeps communication within the project context. |
| **Version Control (Sent Quotes)** | Track specific versions of quotations sent to clients. If a quote is renegotiated, a new version is created while preserving the history of what was previously sent. | **Medium** | Provides a clear audit trail of what was promised and agreed upon at different stages. |
| **Multi-Currency Support** | Ability to display costs in different currencies, with manual or automatic exchange rate conversion for international clients. | **Low** | Critical for agencies working with international clients to present costs in the client's local currency. |
